# 🚀 CareerPilot AI

### AI-Powered Multi-Agent Career Intelligence Platform

CareerPilot AI is a modern AI-powered career guidance platform that helps students from **any academic background** make smarter career decisions through specialized AI agents.

---

# 🌟 Features

- 🎯 Personalized Career Guidance
- 📄 AI Resume Analyzer
- 🗺️ Learning Roadmap Generator
- 🎤 Interview Preparation
- 📊 Skill Gap Analysis
- 🧠 Multi-Agent AI Architecture
- 📁 PDF Resume Upload
- 🤖 Agent Activity Tracking
- ⚡ FastAPI Backend
- 💻 React Frontend

---

# 🚀 Getting Started

## Backend

```bash
cd backend

pip install -r requirements.txt

uvicorn app.main:app --reload
```

---

## Frontend

```bash
cd frontend

npm install

npm run dev
```
