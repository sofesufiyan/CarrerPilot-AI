# CareerPilot-AI

## Project Vision

CareerPilot-AI is an AI-powered career mentor and learning companion that helps students learn AI, build projects, prepare for interviews, improve communication, create resumes, discover internships, and achieve their career goals.

The application combines Google's Agent Development Kit (ADK), Gemini, FastAPI, MCP, and modern web technologies to provide personalized guidance.

---

# Problem Statement

Many students struggle with:

* Choosing the correct career path
* Learning AI systematically
* Building industry-level projects
* Finding internships
* Preparing resumes
* Improving interview skills
* Tracking learning progress
* Discovering hackathons and certifications

CareerPilot-AI acts as an intelligent AI mentor that provides personalized guidance throughout the student's learning journey.
